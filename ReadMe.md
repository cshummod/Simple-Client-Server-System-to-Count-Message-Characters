#	Message Characters Counter using TCP Socket   version(1.0)

<b>This is a simple program written by python. which receive message from 
client and response with message characters count.</b>

### Requirements
		* Any version of Windows/Mac or Linux OS
		* Python 3.0 and above	


### Usage
		1. Run Server.py
		2. Run Client.py 
		3. Enter server hostname/IP.
		4. Enter server port.
		5. Enter your message.
		6. The server will reply with message characters count.
